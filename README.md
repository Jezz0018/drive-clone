# Google Drive Clone

A production-ready Google Drive clone built with SvelteKit and FastAPI.

## Features
- Modern UI (TailwindCSS)
- File/Folder Management
- Authentication (JWT)
- Search & Sharing
- Trash, Starred, Recent

## Tech Stack
- **Frontend**: SvelteKit, TailwindCSS, TypeScript
- **Backend**: FastAPI, PostgreSQL, SQLAlchemy
- **Database**: PostgreSQL
